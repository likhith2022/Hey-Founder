# License Compliance

AI Company OS is an original local-first AI company operating system implementation.

Paperclip was used only as product inspiration unless otherwise stated. Do not use the Paperclip name, logo, screenshots, branding, website copy, or other product assets in AI Company OS materials.

Do not copy Paperclip source code without preserving its MIT license and copyright notice. If MIT-licensed code from Paperclip or any other project is copied or adapted later, add the project, copyright notice, license, source URL, and affected files to `THIRD_PARTY_NOTICES.md`.

Before any public or commercial release:

- Run `corepack pnpm license:check`.
- Run `corepack pnpm compliance:check`.
- Review dependency license output for unknown, GPL, AGPL, LGPL, MPL, EPL, CDDL, or other reciprocal/copyleft terms.
- Manually review README, release notes, website copy, screenshots, demo scripts, UI text, comments, prompts, and marketing material for branding confusion or copied wording.
- Confirm no Paperclip name, logo, screenshots, branding, or website copy are used.
- Confirm any copied or adapted third-party source code preserves required notices.
