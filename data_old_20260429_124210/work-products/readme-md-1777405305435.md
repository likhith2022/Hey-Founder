# Prajwal Sharma's Portfolio - NovaStack AI Landing Page

This archive contains the core assets for a landing page developed for NovaStack AI, a software startup building practical AI productivity tools for revenue teams. This project demonstrates foundational web development skills, including HTML structure and CSS styling.

## Project Overview

The NovaStack AI landing page is designed to introduce the company and its offerings. It features a responsive layout, clear call-to-actions, and sections for features, testimonials, and an 'About Us' section.

## Files Included

*   `index.html`: The main HTML file, structuring the content of the landing page.
*   `style.css`: The CSS stylesheet, defining the visual presentation and layout of the landing page.
*   `README.md`: This file, providing an overview of the project and its contents.

## How to View

To view the landing page, simply open the `index.html` file in any modern web browser.

*   HTML5
*   CSS3
*   Google Fonts (Inter)
